# Drum Kit

![App preview of Drum Kit](images/drum-kit-demo.gif "Drum Kit Demo")

## :wave: Intros!

Drum Kit is a small web app that features a list of playable keys that renders different types of drum sound effects.

Check out the app and get "beat" on! Just don't forget those nice little feedbacks :)

## :pushpin: Project Objectives

* Create a list of audios with their attached drum sounds.

* Create a list of keys that corresponds to keyboard inputs.

* Link the drum sounds and input keys together. When the keys are pressed, it plays the corresponding sound(s).

* Layout and style the app with clean & well-formatted CSS. Nothing complex => **_KISS_**

## :gear: Technical Components

_#### This is a Frontend-only web app!_ 
* Basic CSS Styles & Selectors

* ES6+ JS DOM Manipulations and Event Handlings
